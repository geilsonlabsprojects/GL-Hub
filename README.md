## 📱 Sobre

O **GL Hub** é um aplicativo Android desenvolvido para centralizar todo o ecossistema da **Geilson Labs Projects**.

Com ele, os usuários podem:

- 📦 Descobrir novos aplicativos
- 🔄 Atualizar aplicativos instalados
- 🌐 Acessar sites oficiais
- 📰 Visualizar novidades
- ⭐ Favoritar aplicativos e sites
- 📜 Consultar o histórico de versões (Changelog)
- 🔍 Pesquisar aplicativos rapidamente
- 📲 Instalar APKs diretamente pelo aplicativo

## ☁ Firebase

Este projeto utiliza exclusivamente recursos compatíveis com o plano gratuito (Spark).

Serviços utilizados:

- Firebase Authentication
- Firebase Realtime Database
- Firebase Remote Config

Não utiliza:

- Firebase Cloud Messaging (FCM)
- Firebase Storage
- Firestore
- Cloud Functions

---

## 📦 Distribuição

Os APKs oficiais são distribuídos através do GitHub Releases.

O aplicativo verifica periodicamente a existência de novas versões e permite a atualização diretamente pelo Android.
